###Consolex
#####*console extension with some color methodes*

###Installation

Install with [npm](http://npmjs.org):

```bash
  npm install consolex --save
```

###Usage
```js
  require('consolex');
  console.yellow("yellow");
  console.red("red");
```

For more usage ,please see `examples/examples.js`.
